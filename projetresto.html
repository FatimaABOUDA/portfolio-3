Projet R3st0.fr

Présentation du Contexte

R3st0.fr est un site web de critique de restaurant dont l'objectif est de recenser les avis des consommateurs et de diffuser ces avis aux visiteurs. Ce site est développé en PHP en suivant le pattern de conception "Modèle-Vue-Contrôleur" (MVC). L'architecture choisie permet une programmation web structurée et modulaire.

Objectifs

Affichage de la Liste des Restaurants : Fournir une interface conviviale pour afficher la liste des restaurants disponibles.

Affichage des Détails des Restaurants : Permettre aux utilisateurs de visualiser les informations détaillées d'un restaurant spécifique.

Recherche de Restaurants : Offrir une fonctionnalité de recherche pour trouver des restaurants basés sur divers critères.

Connexion Utilisateur : Implémenter un système de connexion pour authentifier les utilisateurs.

Gestion des Restaurants Favoris : Permettre aux utilisateurs connectés de marquer des restaurants comme favoris ou non favoris.

Visualisation des Notes : Afficher les notes et avis laissés par les utilisateurs pour les restaurants.

Technologies Utilisées

PHP : Langage de programmation côté serveur pour gérer la logique de l'application.

MySQL : Système de gestion de base de données pour stocker les informations sur les restaurants et les avis.

HTML/CSS : Langages pour structurer et styliser les pages web.

MVC (Modèle-Vue-Contrôleur) : Pattern de conception pour séparer la logique de l'application en composants distincts.
