Projet le petit jardinier

Contexte du Projet :

Ce projet vise à développer une application web pour la gestion de devis et d'informations sur les haies. L'objectif principal est de fournir une plateforme où les utilisateurs peuvent créer, consulter et gérer des devis pour la plantation ou la taille de haies, ainsi que visualiser des informations sur différents types de haies disponibles.

Objectifs :

  • Gestion des Devis : Permettre aux utilisateurs de créer des devis pour la plantation ou la taille de haies en fournissant des informations telles que la longueur, la hauteur et le type de haie.
  • Visualisation des Haies : Afficher des informations sur différents types de haies disponibles, y compris leur nom, leur prix unitaire et d'autres détails pertinents.
  • Authentification des Utilisateurs : Mettre en place un système d'authentification pour permettre aux utilisateurs de s'inscrire, de se connecter et de gérer leur profil.
  • Gestion des Profils Utilisateurs : Permettre aux utilisateurs de visualiser et de mettre à jour leurs informations personnelles, y compris leur nom, leur adresse et leur mot de passe.
  • Gestion des Rôles Utilisateurs : Définir différents rôles d'utilisateur (par exemple, client, administrateur) avec des autorisations distinctes pour accéder aux fonctionnalités de l'application.
  • Interface Utilisateur Conviviale : Concevoir une interface utilisateur intuitive et conviviale pour faciliter la navigation et l'utilisation de l'application.

Technologies Utilisées :

  • Symfony : Un framework PHP utilisé pour le développement backend de l'application. Symfony fournit des composants puissants pour la création rapide d'applications web robustes et évolutives.
  • Doctrine : Une bibliothèque de mapping objet-relationnel (ORM) utilisée pour interagir avec la base de données et gérer les entités dans l'application Symfony.
  • Twig : Un moteur de template utilisé pour générer des vues HTML dynamiques basées sur les données fournies par le contrôleur Symfony.
  • HTML/CSS/JavaScript : Utilisés pour développer l'interface utilisateur frontend de l'application et fournir une expérience utilisateur interactive et attrayante.
  • MySQL : Une base de données relationnelle utilisée pour stocker les données relatives aux utilisateurs, aux devis et aux informations sur les haies.
