Projet Gestion de Ressources

Contexte

Le projet de développement d'une base de données s'inscrit dans le cadre de la gestion des ressources pour la publication d'un livre, visant à offrir à un auteur un outil efficace pour organiser ses données et ses collaborations.

Objectifs du Projet

  • Conversion de Fichier Word en XML : Extraire les données du document Word de l'auteur et les convertir en format XML pour une manipulation informatique ultérieure.
  • Importation dans Excel : Importer les données XML dans Microsoft Excel pour une visualisation et une manipulation aisées.
  • Création de la Base de Données : Concevoir et implémenter la structure de la base de données en identifiant les entités et les relations nécessaires pour stocker les informations sur les auteurs, les documents, les sources, les lieux et les types de documents.
  • Gestion des Ressources : Permettre à l'auteur de gérer ses données, y compris les détails sur les auteurs impliqués, les documents associés, les sources citées, les lieux pertinents et les types de documents utilisés.
  • Implémentation MVC : Organiser le projet selon l'architecture Modèle-Vue-Contrôleur (MVC) pour une séparation claire entre la logique métier, la présentation et le contrôle des données.

Technologies Utilisées

  • PHP : Langage de programmation pour la logique métier côté serveur.
  • MySQL : Système de gestion de base de données pour stocker et manipuler les données de l'application.
  • Modèle-Vue-Contrôleur (MVC) : Architecture de conception pour organiser le projet de manière modulaire et structurée.
  • Microsoft Excel : Outil de feuille de calcul pour l'importation et la visualisation des données.
  • HTML/CSS : Langages de balisage et de style pour la création de l'interface utilisateur.
  • Visual Studio Code : Environnement de développement intégré pour l'écriture du code.
  • phpMyAdmin : Interface web pour la gestion de la base de données MySQL.
  • WampServer : Ensemble de logiciels pour le développement web local.
