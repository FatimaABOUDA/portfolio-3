Projet de Gestion d'Interventions

Introduction

Le projet de gestion d'interventions est une application web permettant de gérer les informations relatives aux interventions réalisées pour les clients. L'application est développée en utilisant le langage PHP pour la logique côté serveur et MySQL pour le stockage des données. L'objectif principal de cette application est de fournir un outil centralisé pour enregistrer, suivre et gérer les interventions effectuées pour les clients.

Configuration

Pour configurer l'environnement de développement et utiliser l'application, les éléments suivants sont nécessaires :

  • Serveur Distant : Un serveur distant est nécessaire pour héberger l'application web et la base de données. Il peut s'agir d'un serveur dédié, d'un serveur virtuel ou d'un service d'hébergement.
  • WinSCP : WinSCP est un logiciel de transfert de fichiers sécurisé (SFTP, SCP et FTP) pour Windows. Il est utilisé pour transférer les fichiers de l'application entre votre machine locale et le serveur distant.

Les Étapes

L'application se compose de plusieurs pages et fonctionnalités pour faciliter la gestion des interventions. Voici une brève description des principales étapes :

  • Page d'Accueil (index.php) : La page d'accueil est le point d'entrée de l'application. Elle redirige vers les différentes pages de recherche, d'ajout et de devis.
  • Ajout d'Intervention : Cette fonctionnalité permet aux utilisateurs d'ajouter de nouvelles interventions en remplissant un formulaire. Le formulaire récupère les informations relatives à chaque intervention, telles que les informations sur le client, le type de service, le matériel, etc. Les données sur le matériel et les services sont récupérées depuis la base de données pour fournir une liste déroulante.

Objectifs

Les objectifs principaux de l'application de gestion d'interventions sont les suivants :

  • Centraliser les Informations : Fournir un emplacement centralisé pour enregistrer toutes les informations relatives aux interventions clients.
  • Faciliter le Suivi des Interventions : Permettre aux utilisateurs de suivre facilement les interventions effectuées, y compris les détails sur les clients, les types de service et le matériel utilisé.
  • Optimiser les Processus : Simplifier les processus de gestion des interventions en automatisant certaines tâches et en fournissant des fonctionnalités conviviales pour les utilisateurs.

Technologies Utilisées

  • PHP : Langage de programmation utilisé pour la logique côté serveur.
  • MySQL : Système de gestion de base de données utilisé pour le stockage des données.
  • HTML/CSS : Langages pour structurer et styliser les pages web.
  • WinSCP : Logiciel utilisé pour transférer les fichiers entre la machine locale et le serveur distant.
