Mes projets :

Projet Photos

Ce projet est une application web de gestion de photos développée en PHP. L'application permet aux utilisateurs de visualiser des photos, des articles associés, et des informations sur les photographes. L'application est structurée pour offrir une interface conviviale et modulaire, facilitant la navigation et l'interaction avec les contenus disponibles.

Projet Resto

Le projet de gestion de restaurants vise à fournir une plateforme conviviale permettant aux utilisateurs d'explorer une variété d'établissements de restauration. Avec des fonctionnalités telles que la récupération de données sur les utilisateurs, les types de cuisine, les restaurants, les photos et les critiques, ainsi que la gestion des utilisateurs et des connexions, l'objectif est de créer une expérience complète pour les passionnés de gastronomie.

Projet seminaires

Ce projet est une application de gestion de séminaire développée en PHP, utilisant des fichiers JSON pour stocker les données. Son objectif est de permettre aux participants de s'inscrire aux conférences et aux administrateurs de gérer les inscriptions. Avec des fonctionnalités de chargement/sauvegarde de données, gestion des participants et des conférences, ainsi que des contrôles d'erreurs et de sécurité, cette application offre une solution efficace et conviviale pour la gestion d'un séminaire.

Gestion de la clientéle et leur facture

Ce projet est une application web développée en PHP, avec une base de données MySQL, visant à gérer les informations relatives aux interventions pour les clients. Il offre des fonctionnalités de recherche, d'ajout et de création de devis. L'interface comprend une page d'accueil redirigeant vers trois principales : recherche, ajout et devis. La page d'ajout propose un formulaire pour saisir les détails des interventions, avec des listes déroulantes pour le matériel et les services, préalablement chargées depuis la base de données.

Analim

Ce projet vise à développer une application web pour faciliter l'organisation du congrès [ANALIM] par la Maison des Ligues de Lorraine (M2L). L'objectif est de créer une plateforme flexible permettant de gérer les inscriptions, les sessions, les activités culturelles, la facturation et d'autres aspects liés à l'événement. En collaboration avec d'autres régions françaises, la M2L organise ce congrès à Limoges, et cette application servira à simplifier et optimiser le processus de gestion de cet événement.

Gestion des sources

Ce projet vise à créer une base de données à partir d'un fichier Word, en passant par la conversion en XML puis l'importation dans Excel pour une meilleure visualisation. Ensuite, nous concevrons la structure de la base de données en identifiant les entités, leurs relations et les types de données appropriés. Les principales entités incluent les auteurs, les documents, les lieux, les sources et les types de documents. Le projet est organisé selon le modèle MVC (Modèle-Vue-Contrôleur) et implémente des fonctionnalités telles que l'ajout, la modification et la suppression d'auteurs, de documents, de sources, de lieux et de types de documents.

Le p'tit jardinier

Le projet consiste en la création d'une application web en utilisant le Framework Symfony pour la société Le p'tit Jardinier, qui se spécialise dans l'entretien des jardins pour les particuliers et les organisations. L'application permet aux utilisateurs d'obtenir des devis pour la taille de haies.
